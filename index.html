About the Event

Code Hunt is a thrilling treasure hunt competition that challenges participants to solve programming puzzles and uncover clues to progress through various levels. It tests problem-solving skills, algorithmic thinking, and coding proficiency in a fun and competitive environment. Participants will face a series of programming challenges that require creativity and logical reasoning to solve. As they successfully complete each challenge, they will receive clues that lead them to the next level, ultimately aiming to reach the top of the leaderboard. The event is spread across 5 levels and 2 major sections.

Phase 1: Problem Solving

Participants will be given a set of programming challenges to solve within a specified time limit, without external assistance(Ai & Other Softwares).

Phase 2: Clue Hunting

Participants will be allowed to use digital resources to solve Phase 2 Questions and Decode the next clues.

Theme

The event is themed around a mysterious digital world where participants must navigate through code-based challenges to uncover hidden treasures using Manual and Software-based Decoding with the help of programming languages like C & Python.
